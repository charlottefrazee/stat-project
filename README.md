# Project Description

This project investigates the whether or not "defense wins championships" in college football by analyzing defensive statistics and their correlation to team wins using various statistical methods. 

# Data Dictionary

| Field Name | Data Type | Description |
|----------|------|-------------|
| **Team_Year** | Text | Unique identifier combining team name and season year (e.g., "Alabama (SEC)_2013") |
| **Games** | Integer | Total number of games played in the season |
| **Wins** | Integer | Number of games won |
| **Losses** | Integer | Number of games lost |
| **Fumbles_Recovered** | Integer | Number of fumbles recovered by the defense |
| **Defensive_Rank** | Integer | Team's overall defensive ranking (lower number = better defense) |
| **Defensive_Plays** | Integer | Total number of defensive plays |
| **Yards_Allowed** | Integer | Total yards allowed to opponents |
| **Total_TDs_Allowed** | Integer | Total touchdowns allowed to opponents |
| **Avg_Sacks_Per_Game** | Integer | Average number of quarterback sacks per game |
| **Offensive_Rank** | Integer | Team's overall offensive ranking (lower number = better offense) |
